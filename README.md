## Textile Enterprise Financial Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/072c3b68e9f542b4bf2f198698df97f6/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Employee), Finance Staff, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Reimbursement Information Management
6. Announcement Information Management
7. Message Management
8. Salary Management



**Finance Staff Functions:**
1. Change Password
2. Personal Information
3. Reimbursement Information Management
4. Announcement Information Management
5. Revenue Information Management
6. Expense Information Management
7. Salary Management


**Administrator Functions:**
1. Login
2. Personal Center
3. Finance Staff Management
4. Reimbursement Information Management
5. Reimbursement Type Management
6. Announcement Type Management
7. Revenue Type Management
8. Expense Type Management
9. Announcement Information Management
10. Message Management
11. Revenue Information Management
12. Expense Information Management
13. Salary Management
14. Employee Management
